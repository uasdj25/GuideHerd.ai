# Conflict Check Rules

## Extraction Only
Extract names and entities only.

Do not:
- decide whether a conflict exists
- infer relationships beyond the intake facts
- add hidden assumptions

## Capture Fields
- potential_client
- opposing_parties
- related_entities
- deceased_persons
- heirs_or_beneficiaries
- witnesses_or_other_names
- opposing_counsel

## Review Standard
If any name/entity appears, preserve it exactly as stated in the intake record.
